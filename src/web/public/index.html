Installation method

Step 1: Install the package

Pick one of these, then choose your platform below to connect it.

Global install recommended

Puts dollhousemcp on your PATH. Available everywhere.

npm install -g @dollhousemcp/mcp-server

Update later: npm update -g @dollhousemcp/mcp-server

Local project install

Installs into a directory you control. Good for isolating versions or running multiple installs.

mkdir -p ~/mcp-servers && cd ~/mcp-servers
npm install @dollhousemcp/mcp-server

Then point your client config at: node ~/mcp-servers/node_modules/@dollhousemcp/mcp-server/dist/index.js

Auto-install auto-updating

Adds DollhouseMCP to your Claude Desktop config using npx, which always pulls the latest version on startup. Restart Claude Desktop after.

Or download the Desktop Extension static version

Download the .mcpb bundle and open it. Zero terminal interaction, but installs a fixed version that won't auto-update.

Download .mcpb

Or add manually

Add to your config file:

  • macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
  • Windows: %APPDATA%\Claude\claude_desktop_config.json
  • Linux: ~/.config/Claude/claude_desktop_config.json
{
  "mcpServers": {
    "dollhousemcp": {
      "command": "npx",
      "args": ["-y", "@dollhousemcp/mcp-server@latest"]
    }
  }
}

Restart Claude Desktop after saving.

Auto-install

Click to automatically add DollhouseMCP to Claude Code.

Or run in your terminal

Start a new conversation and you're ready.

claude mcp add dollhousemcp -- npx -y @dollhousemcp/mcp-server@latest

Or add manually

Add to ~/.claude.json:

{
  "mcpServers": {
    "dollhousemcp": {
      "command": "npx",
      "args": ["-y", "@dollhousemcp/mcp-server@latest"]
    }
  }
}

Or add config manually

Add to .cursor/mcp.json in your project, or ~/.cursor/mcp.json for all projects:

{
  "mcpServers": {
    "dollhousemcp": {
      "command": "npx",
      "args": ["-y", "@dollhousemcp/mcp-server@latest"]
    }
  }
}

Or configure via Settings > MCP Servers in the Cursor UI.

Or add workspace config manually

Add to .vscode/mcp.json in your workspace:

{
  "servers": {
    "dollhousemcp": {
      "command": "npx",
      "args": ["-y", "@dollhousemcp/mcp-server@latest"]
    }
  }
}

VS Code uses "servers", not "mcpServers".

Or run in your terminal

Run this in your terminal:

codex mcp add dollhousemcp -- npx -y @dollhousemcp/mcp-server@latest

Or add to config

Add to ~/.codex/config.toml (Codex uses TOML, not JSON):

[mcp_servers.dollhousemcp]
command = "npx"
args = ["-y", "@dollhousemcp/mcp-server@latest"]

Or run in your terminal

Run this in your terminal:

gemini mcp add dollhousemcp -- npx -y @dollhousemcp/mcp-server@latest
